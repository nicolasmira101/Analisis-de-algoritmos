El código fuente del algoritmo desarrollado por el equipo se encuentra en el archivo BullsAndCows_proyecto.py 

Para ejecturar el algoritmo desde Linux escribir en consola:

1. python3 PlayBullsAndCows.py #caracteres diccionario BullsAndCows_proyecto BullsAndCows_proyecto

Por ejemplo, si se desea jugar con 4 posiciones y un abecedario de las letras a la j entonces la instrucción sería la siguiente:

- python3 PlayBullsAndCows.py 4 abcdefghij BullsAndCows_proyecto BullsAndCows_proyecto
